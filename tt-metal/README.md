# TT-Metal docs

Notes and reference material related to TT-Metal usage and kernel development.

Files:
- `kernel-build-and-cache.md`: JIT pipeline, toolchain, cache layout, disassembly tips.
- `kernel-loading-and-abi.md`: XIP packing, kernel-config layout, runtime args ABI, add1 trace.
- `dispatch-modes.md`: fast vs slow dispatch, architecture mapping, harvesting implications.
- `dataflow-and-cbs.md`: CB semantics, dataflow patterns, buffer copies, L1-direct vs DRAM.
- `sfpi-and-kernel-dev.md`: SFPI mental model, add1 SFPI kernel, dst/srca staging.
- `debug-env-vars.md`: debugging and profiling env vars.
